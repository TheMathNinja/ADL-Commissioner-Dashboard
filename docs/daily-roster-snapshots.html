Purpose

Daily Roster Snapshots exist to provide a consistent historical record of ADL roster movement for commissioner accounting purposes. This dashboard's script runs automatically each day at 5 a.m. Eastern, and also during manual or push-triggered runs. A new dated CSV is published only when the roster data changes from the latest saved snapshot. The snapshots include roster, salary, contract, franchise, conference, and roster status information.
